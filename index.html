Специалист по тестированию ПО в 2GIS. Команды Web API и Continuous Delivery.

Занимаюсь задачами внедрения Continuous Integration и Continuous Delivery в Web API, автоматизацией для разработки, функционального и нагрузочного тестирования.

Цель

Всегда должен быть рабочий код.

Проблемы

  • Выпуск новой задачи с дефектами
  • Дефекты в продукте из-за новой задачи
  • Неожиданности под нагрузкой

Продукт
не стабилен.

Continuous Integration

  • Автоматизированное разворачивание
  • Автотесты
  • Быстрое выполнение
  • Тестирование на production-like окружении
  • Работоспособность мастер-ветки и системы в приоритете
